核心理念
激进但**正期望**。小本金求快速积累,不靠赌——靠"选对票 + 选对策略 + 严格出场纪律"把期望值做正,活着复利。期望值比胜率更重要。
一、先诊断,再决定打法(核心)
不同股票性格不同,用错策略=送钱。回测把每只票分成三类,卡片上有标签:
📉 宜抄底:会干净回踩50日线的票(CIEN/AMAT/SMCI/OKLO/IBM…)→ 缠论二买/回踩买。
📈 宜突破:只冲不回踩的动量龙头(NVDA/AMD/MU/WDC/RKLB…)→ 等突破前高追,别抄底。
🚫 回避:波动太大被洗(MRVL/CRDO/ARM…)或抛物线崩塌(NVTS/UUUU…)→ 不碰或等VCP紧致基底。
二、三种策略怎么打
策略用于入场出场止损
📉抄底宜抄底票回踩50MA+当日反转,分批3档半仓+2R锁胜,半仓跟50MA跑近5日低
📈突破宜突破票突破近30日高(pivot)跟50MA跑基底下沿(高波动票用~2×ATR宽止损+半仓)
🏔️长持主动交易反复亏的长牛(MRVL/ARM…)正股回踩50/150/200日线越跌越买前高分批止盈破200日线离场·绝不用杠杆/期权·扛大回撤
🚫回避回避票不入场(或等VCP转突破)——
三、出场纪律(回测最大发现)
诊断三问发现两大漏点:①卖太早(卖出后20日还涨17.6%)②止损太紧(72%假止损后又收复)。
对策:让赢家跟50日线跑(期望翻倍)+ 分批(半锁胜半吃趋势)+ 假止损后收复入场价再进场。
四、6条激进铁律
① 别买深价外近月call,用贴价/价内或价差。
② 只在IV分位<40买裸call;IV贵(>70)的看涨用正股/价差。
③ call翻倍先卖回本金,让利润跑。
④ 单笔风险≤8%;一条题材算一个仓位≤60-70%;留20%+干火药。
⑤ 只追Stage2且回调买,不追新高,Stage4不接飞刀。
⑥ 搏高赔率用:便宜IV长call / 定义风险的价差 / 明确催化事件。
五、每日流程
1. python run.py build 刷新全部数据(或起 serve.py 用页内🔄按钮)。
2. 自选按每票策略标签找信号:宜抄底看回踩买点,宜突破看突破触发位。
3. 下单前对照 6 铁律 + 仓位风险(单笔≤8%)。
4. 收盘后:今日推荐已冻结,逐日前向复盘"点位准不准"。
