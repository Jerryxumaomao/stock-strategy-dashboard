# A股量化知识手册(ashare 分支)

> 给在 A 股使用本工具的朋友。**核心一句话:美股的动量/追涨逻辑在 A 股会失效甚至反向——A 股短期是均值回归(反转)市场。** 照搬美股策略会亏钱。本文讲清 A 股和美股在量化上的根本差异、A 股独有的因子与理论、每个策略该怎么改、以及数据从哪来。
> 所有内容为决策辅助,非投资建议;历史规律不保证未来。

---

## 一、为什么 A 股必须单独一套(市场结构差异)

| 维度 | 美股 | A 股 | 对量化的影响 |
|---|---|---|---|
| **涨跌停** | 无(熔断除外) | 主板 ±10%、创业板/科创板 ±20%、北交所 ±30%、ST ±5% | **突破策略半失效**:涨停封死("一字板")时你根本买不进;跌停封死时止损单成交不了。不能把涨停当"可成交的突破点" |
| **交易制度** | T+2 交割、可日内回转 | **T+1**:今天买的明天才能卖 | 止损无法当日执行,最快次日开盘。日内策略不可用。回测若按 T+0 会高估收益 |
| **做空** | 可自由做空 | 融券极贵/难,散户基本 **只能做多** | "回避档"= 不买(不能靠做空赚下跌);阿尔法只能来自多头选股 |
| **交易时间** | 9:30–16:00 连续 | 9:30–11:30 + 13:00–15:00(午休),9:15–9:25 **集合竞价** | 集合竞价定开盘价;打板资金在竞价阶段博弈 |
| **参与者结构** | 机构主导 | **散户占成交 ~60%+** | 情绪驱动强、题材炒作、换手率高、短期反转显著 |
| **停牌** | 罕见 | 较常见(重组/核查) | 停牌期数据断层,回测要处理 |

**结论**:A 股是一个"高波动、长期只能做多、短期均值回归、情绪与题材驱动、有价格天花板"的市场。它有一整套美股没有的数据(龙虎榜、北向、涨停板)和因子(反转、换手、情绪周期)。

## 二、A 股独有的量化因子与理论

### 1. 短期反转效应(反转因子)—— 与美股动量相反,A 股最稳的 alpha 之一
- **现象**:过去 ~1 个月的**输家**下月倾向反弹,**赢家**倾向回吐。和美股的"动量延续"(Jegadeesh & Titman 1993)**方向相反**。
- **为什么**:散户追涨杀跌造成过度反应,机构少、纠错慢;流动性好+高换手放大了短期错杀。
- **学术**:A 股短期反转是被反复验证的因子(如 Cheema & Nartea 等对中国市场反转的研究;国内券商金工"短期反转因子"是标配)。**中长期(6–12 月)动量在 A 股很弱甚至不存在**——这点和美股差异巨大。
- **本工具实现**:`lab/ashare.py::reversal_score` —— 过去 21 日跌得多 → 反转分高(超跌反转候选);涨得多 → 提示"A 股短期易回吐,勿追"。**这是把美股的"突破追涨"降权、"超跌低吸"升权的核心依据。**

### 2. 小市值因子(及其 2017 后的衰减)
- **现象**:2016 年前 A 股小市值股长期大幅跑赢(壳价值:借壳上市预期给小盘"壳溢价")。
- **转折**:2017 年后 IPO 常态化 + 退市趋严 + 注册制 → **壳价值瓦解,小市值因子大幅衰减甚至反转**,龙头/大盘风格阶段占优。
- **用法**:不要无脑买小盘;市值风格是**轮动**的(中证1000 vs 沪深300 相对强弱)。别用一个 2015 年的小盘策略在今天跑。

### 3. 换手率 / 流动性因子
- **现象**:A 股散户多、换手率远高于美股。**极高换手常是局部顶部(拥挤、接盘)**,极低换手=被忽视(可能是价值洼地或将死票)。
- **本工具实现**:`turnover_stats` —— 换手率历史分位;≥90 分位提示"拥挤见顶风险",≤10 分位"关注度不足"。

### 4. 北向资金(沪深股通)—— "聪明钱"情绪计
- **现象**:外资通过陆股通进出,被广泛当作机构/聪明钱风向标;北向大幅净流入常伴随白马蓝筹走强。
- **注意**:近年北向的"聪明钱"叙事被质疑(含大量交易型资金),**参考不迷信**。
- **本工具实现**:`northbound_flow`(akshare)。

### 5. 龙虎榜(游资席位追踪)—— A 股独有的资金透明窗口
- **现象**:每日盘后披露买卖前 5 营业部席位。知名游资席位(如"作手""章盟主"等)、机构专用席位的动向可追踪。**打板/接力体系的核心数据。**
- **本工具实现**:`dragon_tiger`(akshare `stock_lhb_detail_em`)。

### 6. 涨停板 / 打板体系 —— A 股最活跃的散户量化流派
- **概念**:首板 / 连板(2 板、3 板…)/ 龙头 / 断板 / 炸板 / 打板成功率 / 连板高度。
- **逻辑**:强势资金用涨停表达态度;龙头股的"连板高度"是市场情绪的温度计。**但打板是高风险博弈**(次日大概率高开低走/炸板),需严格纪律。
- **本工具立场**:提供**情绪数据**(见下),但**不鼓励无脑打板**;涨停在本工具里主要用于**识别不可成交点**(别把封死的涨停当突破买点)和**情绪判断**。

### 7. 市场情绪周期 —— A 股择时的独特维度
- **周期**:冰点 → 修复 → 发酵 → 高潮 → 退潮(循环)。
- **体温计**:**涨停家数、最高连板高度、炸板率、跌停家数**。炸板率高企 + 连板高度崩塌 = 退潮,风险最大;冰点(涨停极少、跌停多)后往往是修复起点。
- **本工具实现**:`sentiment_gauge`(akshare 涨停池/炸板池)。**A 股择时看情绪周期,美股择时看 SPY 200 日线——完全不同的东西。**

## 三、每个策略在 A 股怎么改(适配指南)

| 原策略(美股) | A 股适配 |
|---|---|
| **突破追涨 (brk/brk_atr)** | **降权**。A 股短期反转,追突破次日常被砸;且涨停封死买不进。只在**非涨停、放量、有情绪配合**时轻仓,严格次日止损(T+1 已让你被动隔夜) |
| **回踩低吸 (dip)** | **升权**。叠加**反转因子**:超跌 + 高换手回落 + 缠论底背驰 = A 股最舒服的低吸位。这是 A 股主战场 |
| **长持 (hold)** | 适用于真龙头(白马蓝筹),但注意 A 股牛短熊长、波动更大,回撤容忍度要更高;分批越跌越买 |
| **回避 (avoid)** | = **不买**(A 股散户不能做空);别接 ST/退市风险票、别接情绪退潮的连板龙头 |
| **大盘开关 (SPY 200MA)** | 改用 **沪深300 (000300) 或中证全指 200 日线** + **情绪周期**双重择时。情绪退潮期即使指数在线上也要减仓 |
| **状态灯(波动前兆)** | 保留,但阈值需用 A 股数据重估(A 股波动更大);叠加涨跌停状态(封板=特殊态) |
| **期权** | A 股个股无期权(只有 50ETF/300ETF/500ETF/1000ETF 期权 + 少量商品期权);个股期权模块**不适用**,可做 ETF 期权但另需接口 |
| **回测** | ⚠️**最大坑**:必须建模 **T+1**(买入次日才可卖)+ **涨跌停**(封死不可成交)。当前引擎继承自美股(T+0 可成交假设),**A 股回测期望值会高估**——见下方"已知局限" |

## 五、每日怎么用(A 股版工作流)

1. **择时先行**:看沪深300 是否在 200 日线上 + 情绪周期(涨停家数/连板高度/炸板率)。退潮期空仓/轻仓,别硬做。
2. **选股**:反转因子(超跌 + 高换手回落)+ 缠论底背驰 → 低吸候选;综合评分档位过滤。
3. **看资金**:龙虎榜有没有游资/机构进场;北向近几日净流入还是流出。
4. **纪律**:T+1 意味着你**当天买错要扛一夜**,所以**买点更要苛刻**;涨停别追(尤其一字);单票仓位控制(A 股波动大)。
5. **回避**:ST/退市风险、情绪退潮的高位连板、无量阴跌的弱势票。

## 六、已知局限(诚实条款,务必读)

1. **回测引擎仍是美股口径**:当前 `lab/backtest.py` 假设 T+0 可当日成交、无涨跌停约束。**A 股实际 T+1 + 涨跌停会让实盘可成交性低于回测**,故回测期望值**偏乐观**。用它做**相对比较**(哪个策略/哪只票相对强)可以,别把绝对期望值当真。T+1+涨跌停的完整回测建模是路线图 TODO。
2. **反转因子是先验**:`reversal_score` 的权重基于文献与经验,未在本工具内做 A 股前向验证;用复盘系统(`run.py review`)积累后再调。
3. **龙虎榜/北向/情绪需中国可达网络**(东财);海外/VPN 环境常被挡,函数会优雅返回错误提示而非崩溃。
4. **缠论在 A 股是原生的**(缠中说禅本就交易 A 股),可直接用;但个股期权不适用(A 股个股无期权)。
5. **小市值/风格因子会漂移**:2017 年注册制后 A 股风格大变,别用旧假设。
6. 所有输出为决策辅助,非投资建议;下单由你手动执行(本工具被禁止碰下单接口)。
